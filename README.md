## Installation

```sh
$ npm install -g ironmaker
```

## Quick Start

Create an app:

```bash
$ ironmaker
```

Install dependencies:

```bash
$ npm install
```

Start the app at `http://localhost:3000/`:

```bash
$ npm start
```
